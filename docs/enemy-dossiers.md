# Enemy Dossiers

> Goal: One file per enemy. Each dossier is actionable: tells → risk → counter.

## Core Rules (confirmed)
- Mask works on most enemies, but NOT on Foxy or the Puppet.
- Puppet is controlled by keeping the music box wound (Prize Room camera).
- Balloon Boy does not kill; he disables the flashlight.
